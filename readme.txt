1.clone the project

    git clone https://github.com/aspark-rupesh/url-shortner.git

    cd to the proj directory (cd url-shortner)

2. Setup a virtual env
  python3 -m venv venv

3. Activate the env
    source venv/bin/activate (for ubuntu)

4. Install Requirements
   pip install -r requirements.txt

5. Run the server 
    python manage.py runserver 0.0.0.0:8000
